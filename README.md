# Front End Test

A starter repository for vue 3, vite, tailwindcss, and unplugin-icons

## Installation

```sh
npm install
```

## Development

```sh
npm run dev
```

## Build

```sh
yarn build
```

## How to use icon

Use `<i-{collection}-{icon}>` in template to autoload icon.

```vue
<template>
    <i-mdi-account />
</template>
```

The full list of collections and icons can be browsed here: https://icones.js.org
